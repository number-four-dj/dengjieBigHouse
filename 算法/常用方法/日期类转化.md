## String 转 Date

```java
  String dateString = "2021-09-24 12:13:14"; // 要转换的日期字符串

  // 方式一
  SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss"); // 定义日期格式
  try {
      Date date = sdf.parse(dateString); // 转换为 Date 对象
      System.out.println("转换后的日期: " + date);
  } catch (Exception e) {
      e.printStackTrace();
  }

  // 方式二
  DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss"); // 定义日期格式
  LocalDateTime localDateTime = LocalDateTime.parse(dateString, formatter); // 转换为 LocalDateTime 对象
  System.out.println("转换后的 LocalDateTime: " + localDateTime);
```

注意，使用 `SimpleDateFormat`的 `parse`方法，需要`try-catch`捕获异常

## Date 转 String

```java
// 方式一
Date myDate = new Date(); // 获取当前日期时间
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss"); // 定义日期格式
String dateString1 = sdf.format(myDate); // 将 Date 转
System.out.println("转换后的字符串: " + dateString1);

// 方式二
LocalDateTime now = LocalDateTime.now(); // 获取当前日期时间
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss"); // 定义日期格式
String dateString2 = now.format(formatter); // 将 LocalDateTime 转换为 String
System.out.println("转换后的字符串: " + dateString2);
```

## **LocalDateTime获取时分秒**

```java
LocalDateTime dateTime =LocalDateTime.now();//获取当前日期时间

int year = dateTime.getYear();
Monthmonth = dateTime.getMonth();
int day = dateTime.getDayOfMonth();
int hour = dateTime.getHour();
int minute = dateTime.getMinute();
int second = dateTime.getSecond();

System.out.println("年: " + year);
System.out.println("月: " + month.getValue());
System.out.println("日: " + day);
System.out.println("时: " + hour);
System.out.println("分: " + minute);
System.out.println("秒: " + second);
```

## **LocalDateTime日期+1**

```java
LocalDateTime dateTime = LocalDateTime.now(); // 获取当前日期时间
// 月份+1月，如果是12月，可能会影响到年的值
LocalDateTime newDatePlus = dateTime.plusMonths(1);
// 日期-10天
LocalDateTime newDatePlusMinus = newDatePlus.minusDays(10);

System.out.println(newDatePlusMinus.format(DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss")));
```

## 获取当天的23点59分59秒

```java
  // 获取当前日期
  Calendar calendar = Calendar.getInstance();

  // 设置时间为今天的23点59分59秒
  calendar.set(Calendar.HOUR_OF_DAY, 23);
  calendar.set(Calendar.MINUTE, 59);
  calendar.set(Calendar.SECOND, 59);
  calendar.set(Calendar.MILLISECOND, 999); // 可选，设置毫秒为999

  // 获取最终的日期对象
  Date endOfDay = calendar.getTime();

  // 输出结果
  System.out.println("当天的23点59分59秒: " + endOfDay);
```
