## 数组排序

```java
public class ArraySortDescending {
    public static void main(String[] args) {
        Integer[] arr = {9, 8, 7, 2, 3, 4, 1, 0, 6, 5};
        
        // 对数组进行降序排序
        Arrays.sort(arr, Collections.reverseOrder());
        
        // 打印排序后的数组
        for (int value : arr) {
            System.out.print(value + " ");
        }
    }
}
```

## 集合排序

```java
public class SortExample {
    public static void main(String[] args) {
        List<Integer> numbers = new ArrayList<>();
        numbers.add(5);
        numbers.add(2);
        numbers.add(8);
        numbers.add(1);

        // 方式1：
        // 使用 Collections.sort() 进行排序
        Collections.sort(numbers);
        System.out.println("升序排序后的列表: " + numbers);

        // 方式2：
        // 使用 Lambda 表达式进行降序排序
        numbers.sort((a, b) -> b - a);
        System.out.println("降序排序后的列表: " + numbers);

        // 方式3：
        // 使用 Stream API 进行排序
        List<Integer> sortedNames = numbers.stream()
                .sorted().collect(Collectors.toList());
        System.out.println("升序排序后的列表: " + sortedNames);
    }
}
```
