# Scanner 导包

```java
import java.util.Scanner;

public class ScannerDemo {
    public static void main(String[] args) {
        String s1, s2;

        Scanner sc = new Scanner(System.in);
        System.out.print("请输入第一个字符串：");
        s1 = sc.nextLine();
        System.out.println("输入的字符串是：" + s1);

        System.out.print("请输入第二个字符串：");

        sc.close();
    }
}
```

#  读取各种格式输入的Demo  

## 获取 **["aa","bb","cc"] 这类输入**

**注意：**

```java
import java.util.Scanner;

public class InputArrayExample {
    public static void main(String[] args) {
        // 创建 Scanner 对象
        Scanner scanner = new Scanner(System.in);
        
        // 提示用户输入
        System.out.println("请输入一个数组，例如 [\"aa\", \"bb\", \"cc\"]:");
        
        // 读取用户输入的整个字符串
        String input = scanner.nextLine();
        
        // 处理输入，去掉方括号并分割成数组
        input = input.replaceAll("[\\[\\]\"]", ""); // 去掉方括号和引号
        String[] array = input.split(",\\s*"); // 按逗号分割，并去掉空格
        
        // 输出数组内容
        System.out.println("您输入的数组为：");
        for (String str : array) {
            System.out.println(str);
        }
        
        // 关闭 Scanner
        scanner.close();
    }
}
```

1. `"[\\[\\]\"]"` 代表的正则表达式是 `"[\[\]"]"`
2. `split(",\\s*")` 这个用法很平常，可以去除多余的空格

## 获取  **[aa,bb,cc] , dd 这类输入**

- 第一种方式

```java
import java.util.Scanner;

public class InputArrayAndStringExample {
    public static void main(String[] args) {
        // 创建 Scanner 对象
        Scanner scanner = new Scanner(System.in);
        
        // 提示用户输入
        System.out.println("请输入格式为 [aa,bb,cc], dd 的字符串:");
        
        // 读取用户输入的整个字符串
        String input = scanner.nextLine();
        
        // 使用正则表达式分割字符串，确保只在外部逗号处分割
        String[] parts = input.split("\\],\\s*"); // 按照 "], " 分割
        
        // 处理第一个部分，去掉方括号并分割成数组
        String arrayPart = parts[0].trim(); // 获取数组部分
        String[] array = arrayPart.replaceAll("[\\[\\]]", "").split(",\\s*"); // 去掉方括号并按逗号分割
        
        // 处理第二个部分，作为字符串
        String stringPart = parts[1].trim(); // 获取字符串部分
        
        // 输出结果
        System.out.println("数组内容：");
        for (String str : array) {
            System.out.println(str);
        }
        
        System.out.println("单独的字符串内容：" + stringPart);
        
        // 关闭 Scanner
        scanner.close();
    }
}
```

- 第二种方式

```java
public class InputSplitExample {
    public static void main(String[] args) {
        String input = "[aa,bb,cc], dd";
        // 先定位数组部分的起始和结束位置
        int startIndex = input.indexOf('[');
        int endIndex = input.indexOf(']');
        // 获取数组部分的字符串
        String arrayPartStr = input.substring(startIndex + 1, endIndex);
        String[] arrayElements = arrayPartStr.split(",");
        for (int i = 0; i < arrayElements.length; i++) {
            arrayElements[i] = arrayElements[i].trim();
        }
        // 获取字符串部分，去掉前面的逗号及空格，以及数组部分的内容
        String stringPart = input.substring(endIndex + 2).trim();

        System.out.println("数组部分: " + java.util.Arrays.toString(arrayElements));
        System.out.println("字符串部分: " + stringPart);
    }
}
```



## 获取  **[[aa,bb,cc],[11,22,33],[111,222,333]] 这类输入**

```java
import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

public class MultiDimensionalArrayExample {
    public static void main(String[] args) {
        // 创建 Scanner 对象
        Scanner scanner = new Scanner(System.in);

        // 提示用户输入
        System.out.println("请输入一个二维数组，例如 [[aa,bb,cc],[11,22,33],[111,222,333]]:");

        // 读取用户输入的整个字符串
        String input = scanner.nextLine();

        // 去掉外层方括号
        input = input.replaceAll("\\[\\[", "").replaceAll("\\]\\]", "");

        // 按照 "], [" 分割成子数组
        String[] subArrays = input.split("\\],\\s*\\[");

        // 创建一个列表来存储最终结果
        List<List<String>> result = new ArrayList<>();

        // 处理每个子数组
        for (String subArray : subArrays) {
            // 去掉方括号并分割成元素
            subArray = subArray.replaceAll("[\\[\\]]", ""); // 去掉方括号
            String[] elements = subArray.split(",\\s*"); // 按逗号分割

            // 将子数组添加到结果列表中
            List<String> tempList = new ArrayList<>();
            for (String element : elements) {
                tempList.add(element);
            }
            result.add(tempList);
        }

        // 输出结果
        System.out.println("您输入的二维数组为：");
        for (List<String> list : result) {
            System.out.println(list);
        }

        // 关闭 Scanner
        scanner.close();
    }
}
```
