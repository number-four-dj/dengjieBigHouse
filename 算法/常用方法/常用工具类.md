# String 常用方法

```java
// 获取指定得字符第一个出现索引下标
System.out.println("abcdabcd".indexOf("c")); // 2

// 将字符串转换为字符数组
char[] chars = "abcd".toCharArray();

// 查找字符串中是否包含某字符
System.out.println("abcdabcd".contains("cd")); // true

// 在字符串中用新的字符替换老的字符
String replace = "abcdabcd".replace('a', 'A');
System.out.println(replace);  // AbcdAbcd

// 去掉字符串中首尾空格
System.out.println("  abcdabcd  ".trim()); // abcdabcd
```

# HashMap常用方法

```java
HashMap<String, Integer> hashMap = new HashMap<String, Integer>();
// 增删改查
hashMap.put("a-1", 1);
hashMap.put("b-5", 1);
hashMap.put("c-2", 1);
hashMap.put("d-3", 1);
hashMap.put("e-6", 1);
hashMap.put("f-4", 1);
hashMap.put("a-1", 101); // 修改
Integer a = hashMap.get("a");
System.out.println(a);
hashMap.remove("f");

// 获取所有 key
Set<String> keySet = hashMap.keySet();

// 获取所有 value
Collection<Integer> values = hashMap.values();

// 获取所有的 map
Set<Map.Entry<String, Integer>> entrySet = hashMap.entrySet();

// 展示所有元素
hashMap.entrySet().forEach(System.out::println);

// 修改元素 全都+1
hashMap.entrySet().forEach(map -> map.setValue(map.getValue() + 1));

// 判断是否存在 key or value
boolean bool1 = hashMap.containsKey("a");
boolean bool2 = hashMap.containsValue(1);
```

### Stream常用方法

```java
// 注意要导入的包
import java.util.stream.Collectors;
import java.util.stream.Stream;

// anyMatch 判断存在 “c” key
boolean bool = hashMap.keySet().stream().anyMatch(key -> key.equals("c"));
System.out.println(bool);

// 过滤 filter  收集 collect(Collectors.toList())
List<String> list = hashMap.keySet().stream().filter(key -> key.equals("c")).collect(Collectors.toList());
```

### 排序Comparator

```java
// 排序 Comparator
hashMap.entrySet().stream().sorted(Comparator.comparingInt(map ->
    Integer.parseInt(map.getKey().split("-")[1])
)).forEach(System.out::println);

hashMap.entrySet().stream().sorted(
    (map1, map2) ->
    Integer.parseInt(map2.getKey().split("-")[1]) -
    Integer.parseInt(map1.getKey().split("-")[1])
).forEach(System.out::println);
```

# Character常用方法

```java
Character a = 'a';
Character b = 'A';
Character c = '5';
Character d = '！';

// 判断字符是否是字母或者数字
System.out.println(Character.isDigit(a));  // true
System.out.println(Character.isLetterOrDigit(a));  // true
System.out.println(Character.isLetterOrDigit(d));  // false

// 字符转为小写
System.out.println(Character.toLowerCase(b));   // a
```
